code-server配置C/C++和Python环境

2021-08-23
技术

网页端和本地配置方法大致相同 思路就是安装相应语言的支持扩展 以下内容只面向新手

C/C++:

首先需要工具包

然后安装相应的vscode扩展 可选安装code runner 这个扩展可以一键运行 当然直接安装的的话会出现不兼容的情况 点击转到下载页，到GitHub下载历史版本 打开vscode的关于，查看VScode的版本 去找支持1.57.1版本的cpp-tools 右键迅雷下载，下载完成之后 从文件夹直接拖到网页版vscode里面，可能会因为服务器带宽原因会卡一下 右键安装即可安装完成 剩下的内容与本地基本一致 gcc对应的C语言文件 g++对应的c++文件 注意事项：在配置以上内容时，一定要打开一个工作目录，路径不要包含中文 因为这几步生成的配置文件都是存在打开的工作目录下，仅对该目录生效

Python

设置与C/C++大致相同 先安装python扩展 再添加配置 在运行之前，根据要运行的文件选择相应的配置
